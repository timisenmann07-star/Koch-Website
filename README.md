# Culinary Kitchen - Unser Schulprojekt

Hey! Wir haben diese Rezept-Website für unser BK2-Projekt gemacht. Sie heißt Culinary Kitchen und ist voll auf Deutsch.

## Was kann man damit machen?

- Rezepte anschauen (wir haben über 30 Stück!)
- Sich anmelden und Rezepte als Favoriten speichern
- Neue Rezepte vorschlagen
- Alles ist responsive, also geht's auch auf dem Handy

## Wie haben wir das gemacht?

Wir haben HTML für die Seitenstruktur verwendet, CSS für das Aussehen und JavaScript für die Interaktivität. Firebase haben wir für Login und Datenbank genommen, weil das einfacher ist als einen eigenen Server aufzusetzen.

### Unsere Dateien:
- index.html: Startseite mit Hero-Bereich
- styles.css: Alles schön machen
- favorites.js: Favoriten hinzufügen/entfernen
- auth.js: Login und Registrierung
- rezepte/: Alle Rezeptseiten

## Firebase Setup

Wir haben ein Firebase-Projekt erstellt mit Auth und Firestore. Die Konfiguration ist in firebase-config.js.

## Wie startet man das?

1. Terminal öffnen
2. `cd` in den HTML-Ordner
3. `python3 -m http.server 8000` eingeben
4. Browser auf http://localhost:8000

## Was war schwer?

- Firebase verstehen (async/await war tricky)
- Responsive Design für alle Geräte
- Datenbank-Struktur überlegen
- Alle Rezepte formatieren

## Was haben wir gelernt?

- Webentwicklung von Grund auf
- Wie man APIs (Firebase) integriert
- Teamarbeit und Debugging
- Warum Kommentare im Code wichtig sind
